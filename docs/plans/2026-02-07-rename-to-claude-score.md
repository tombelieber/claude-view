# Rename: vibe-recall / claude-view ‚Üí claude-score

## Decision

Rename the project from `vibe-recall` (internal) / `claude-view` (npm/repo) to **`claude-score`** everywhere.

**Rationale:** The "AI Fluency Coach" repositioning centers around the AI Fluency Score ‚Äî a shareable number. `claude-score` is instantly understood, SEO-friendly in the Claude ecosystem, and reinforces the viral hook: "What's your Claude score?"

## Naming Map

| Context | Old | New |
|---------|-----|-----|
| npm package | `claude-view` | `claude-score` |
| GitHub repo | `tombelieber/claude-view` | `tombelieber/claude-score` |
| Install command | `npx claude-view` | `npx claude-score` |

## Backwards Compatibility

| Concern | Handling |
|---------|---------|
| `CLAUDE_VIEW_PORT` env var | Keep as fallback: check `CLAUDE_SCORE_PORT` first, then `CLAUDE_VIEW_PORT`, then `PORT` |
| `~/.cache/claude-view/` | Old cache ignored, clean install to `~/.cache/claude-score/` |
| `~/.cache/vibe-recall/vibe-recall.db` | Old DB ignored, fresh index to new path. Users don't store data, it's regenerated from JSONL. |
| Existing `npx claude-view` installs | `npm deprecate` message directs to `npx claude-score` |
