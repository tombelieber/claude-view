## New features

**Multi-provider sessions.** claude-view now ingests sessions from 25 other AI coding agents — Amp, Codex, and more — right alongside Claude Code. One history, one timeline, one place to look, no matter which agent did the work.

**Cross-provider usage analytics.** See token and cost aggregates broken down per provider, so you can compare where your AI spend actually goes.

**Search and export across providers.** Full-text transcript search now spans every provider, and you can export the results.

**Goals on live cards.** A running session's `/goal` shows right on its live card, so you can see what each agent is working toward at a glance.

**Supply-chain malware check.** A built-in session check flags exposure to the active Shai-Hulud / Miasma / Hades npm worm campaign targeting AI-tool developers.

## Improvements

**Accurate cost for Claude Fable 5 and Sonnet 5.** Sessions on the newest Claude 5-family models now show real cost instead of "Unavailable" — Sonnet 5 at its current introductory rate, so the number matches what you're actually billed today.

**No more duplicate Closed cards.** Resuming a session no longer leaves a doubled card in the Closed column.

**Parser keeps up with Claude Code.** Added coverage for Claude Code's newer bridge and background-session records, so a CLI update can't silently slip past the schema guard.

## Getting started

```bash
npx claude-view@latest
```
