## New features

**Prompt origin labels.** User messages now show how each prompt arrived: an "SDK" pill when the Agent SDK injected it, a "system" pill for system-generated prompts like task notifications. Prompts you typed yourself stay clean — no label. It's read straight from the session record, so it appears only when it's certain.

## Improvements

**Releases stay formatter-clean.** Version bumps no longer leave the lint gate drifting from one release to the next.

**Steadier file-watch tests.** Live-streaming tests now tolerate file-system event latency under the larger test suite, so releases don't flake on timing.

**Parser keeps up with Claude Code.** Added coverage for a new Claude Code metadata field so a CLI update can't silently slip past the schema guard.
