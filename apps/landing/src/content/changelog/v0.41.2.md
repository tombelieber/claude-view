## Fixes

**`claude-view stats`, `monitor`, and `live` now render real data instead of just a header.** The three CLI subcommands have been blank since the CLI shipped: they read snake_case JSON keys (`total_sessions`, `cpu_percent`, `session_id`) while the server has always serialized camelCase nested under `currentWeek`, `cost.totalUsd`, and similar. Every key missed silently, every `println!` was skipped, and users saw only `Dashboard Statistics\n--------------------------------------------------` with nothing under it. From v0.41.2, wire-format view structs deserialize with `#[serde(rename_all = "camelCase")]` and any future schema drift surfaces as a user-visible parse error ("server returned unexpected shape — try `--json`") instead of a silent blank.

## Improvements

**`format_count` now handles billions.** Current-period token usage is ~19B for active users; the old K/M-only formatter would have rolled over once `stats` started rendering. Adds a `B` suffix so the usage summary stays readable at scale.
