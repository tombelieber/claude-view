## New features

**Take over a CLI session from the dashboard.** Fork a running terminal Claude session into an interactive one and drive it from the GUI — answer permission prompts, steer the plan, change models — without losing the session's history.

**Trustworthy interaction prompts.** Permission requests never silently auto-deny. The countdown becomes a calm "waiting for you" indicator that pauses while the tab is hidden, and a failed send is surfaced ("couldn't deliver — tap to retry") instead of optimistically clearing. A prompt only resolves once the agent has actually consumed your decision.

**Live monitor first-run empty state.** A polished empty state greets new workspaces with a clear path into your session history, instead of a blank panel.

**Self-maintaining Claude Code compatibility.** claude-view now tracks new Claude Code record and field types automatically, so a Claude Code update can't silently break parsing or hide your sessions.

## Improvements

**No more "runs a few minutes, then quits."** The indexer's periodic re-scan was rewritten from O(N²) to O(N), so workspaces with thousands of sessions stay light on memory instead of climbing until the operating system kills the server.

**The dashboard self-heals.** If the server ever exits unexpectedly, the launcher restarts it automatically (bounded, with an actionable message if it keeps failing) — no more staring at a dead page until you manually restart.

**One hiccup can't take down the whole app.** A panic in a background task is now isolated and logged instead of aborting the entire process.

**Live chat stays responsive under load.** Long conversations no longer re-serialize the whole transcript on every event, and a slow or backgrounded browser tab can't balloon the sidecar's memory.

**New models never show "Unavailable."** Pricing for a brand-new model falls back to the nearest same-family rate, so costs keep showing the moment a new model ships.
